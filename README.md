# Mood Aquarium

An interactive music visualization where songs become living fish in a dynamic ocean ecosystem.

## Features
- Spotify-inspired mood visualization
- Boids-based schooling behavior
- Real-time search filtering
- Interactive audio previews
- Dynamic ocean mood system

## Tech Stack
- Vanilla JavaScript
- SVG animation
- Boids flocking algorithm
- HTML/CSS

## Concept
Each song becomes a fish:
- Valence → color
- Energy → speed
- Tempo → movement style

https://paulagjantunen-hue.github.io/Mood-Activated-SVG-Aquarium/

## Why I built it
To explore how data (music) can be an emotional, visual ecosystem rather than a static list.
